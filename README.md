## ➕ Protege installation process

1. Go to https://github.com/protegeproject/protege
2. On the right side of the page, find the Releases section and click + X releases.
3. Select Protege Platform 5.6.2.
4. In the description, you’ll see the text “To download this release please go here.” Click on here.
5. Under Assets, download the file that matches your operating system.
6. Create a new folder on your computer called Protege.
7. Move the .zip file you downloaded into the Protege folder.
8. Extract the contents of the .zip file.
9. Open the extracted folder (and any subfolders) until you find Protege.exe.
10. Double-click Protege.exe to start the program.
